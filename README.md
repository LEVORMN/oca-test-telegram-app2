# FireGuard Steel Landing Page

Одностраничный продающий сайт по огнезащите металлоконструкций. Проект выполнен на чистом HTML, CSS и JavaScript с современным премиальным интерфейсом, плавными анимациями и адаптивным меню.

## Структура

```
oca-test-telegram-app2/
├── index.html
├── css/
│   ├── variables.css
│   ├── styles.css
│   ├── layout.css
│   ├── components.css
│   ├── sections.css
│   └── responsive.css
├── js/
│   └── main.js
├── assets/
│   └── icons/
└── README.md
```

## Особенности

- 12 продающих секций по формуле AIDA
- Премиальное меню с blur-эффектом, scrollspy и мобильным hamburger
- Glassmorphism карточки, градиенты и анимации
- SVG иконки и изображения из Unsplash
